# CLICON

Fullstack MERN App using React EXPRESS Node TypeScript Jest and MongoDB

![LANDING PAGE](./client/src/assests/01_Homepage.png)

## Stack

- Dev
- Node
- TypeScript
- Mongoose
- MongoDB
- Redux

- Test
  - Jest

## Execute the App

```sh
$ npm install
$ npm start
```

## Test the App

```sh
$ npm run test
```
